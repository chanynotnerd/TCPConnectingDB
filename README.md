# Term1_TCPConnectingDB
---
#### C++를 사용하여 TCP/IP 서버 및 클라이언트 구현, DB와 연동

> 1. 멀티 스레드 활용하여 여러 클라이언트 접속 가능한 채팅서버 구현
> 2. 클라이언트 내에서 quit 입력 시 클라이언트 종료
> 3. 서버에서 각 클라이언트가 지정한 이름으로 유저 구분 가능, 중복 불가능
> 4. DB 연동 및 클라이언트 이름 테이블에 등록
> 5. 상시 수정 중
